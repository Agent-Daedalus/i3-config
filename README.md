# My i3wm Configuration

This repository contains my personal config for the i3 tiling window manager. This configuration is based on the default configuration provided by the i3 config wizard, but has been extensively modified. The configuration provides keybinds for efficient window management and a keyboard-driven workflow, a customized titleless theme, and integration with many common applications. 

## Features
* Wallpaper Integration (using feh)
* Screenshot Hotkeys (using scrot)
* Power Management Hotkeys
* Polybar (Info Bar)
* Picom (Compositor)
* Audio Function Keys
* CMUS (Music Player) Integration
* Quick Launcher
* Neovim & VIFM (CLI Tools) 
* Dual Monitor Support
